Apa itu Struktur Data?

Struktur data adalah cara mengorganisasi dan menyimpan data agar dapat digunakan secara efisien.

  • Efisiensi waktu
  • Efisiensi memori
  • Kemudahan manipulasi data

Contoh Struktur Data

  • Array
  • Linked List
  • Stack
  • Queue
  • Tree
  • Graph

Klasifikasi Struktur Data

  • Struktur Data Linier: Array, Linked List, Stack, Queue
  • Struktur Data Non Linier: Tree, Graph

Abstract Data Type (ADT)

ADT adalah model logis yang mendefinisikan data dan operasi tanpa menjelaskan implementasi.

  • Fokus pada operasi
  • Tidak tergantung bahasa pemrograman

Contoh ADT Stack

  • Push → tambah data
  • Pop → hapus data
  • Peek → lihat data teratas
  • IsEmpty → cek kosong

Konsep: LIFO (Last In First Out)

ADT vs Struktur Data

  • ADT: Konsep logis
  • Struktur Data: Implementasi nyata
